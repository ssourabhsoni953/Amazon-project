# 🛒 Amazon Clone

[![Deploy Status](https://img.shields.io/badge/Deployed_on-Vercel-black?logo=vercel)](#)
[![Tech Stack](<https://img.shields.io/badge/Tech_Stack-HTML5%20%7C%20CSS3%20%7C%20JS(ES6+)-blue>)](#)

A high-performance, fully functional E-commerce web application inspired by Amazon. This project demonstrates core e-commerce logic, including product filtering, shopping cart management, and a seamless checkout flow.

**🚀 Live Demo:** [amazon-fawn-beta.vercel.app](https://amazon-fawn-beta.vercel.app/)

## ✨ Key Features

- **🛍️ Product Catalog:** Dynamic rendering of products with real-time price updates.
- **🛒 Shopping Cart:** Add, remove, and update item quantities with instant subtotal calculations.
- **📦 Order Tracking:** Dedicated "Returns & Orders" section to manage past purchases.
- **📱 Responsive UI:** Optimized for both desktop and mobile users using modern CSS techniques.
- **💾 Persistent Storage:** Maintains your cart data across browser refreshes.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Deployment:** Vercel
- **Tools:** Git & GitHub for version control

## 🚀 Getting Started

**1. Clone the Repository:**

```bash
git clone https://github.com/ssourabhsoni953/Amazon-project.git
```

**2. Open the Project:**
Simply open `amazon.html` in your favorite browser. No build steps required!
